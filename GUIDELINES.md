# 文章清洗工具开发指南

## 项目结构规范

### 文件组织
```
project/
├── index.html      # 主页面
├── styles.css      # 样式文件
├── script.js       # JavaScript逻辑
├── README.md       # 项目说明
└── GUIDELINES.md   # 开发指南
```

## 开发规范

### HTML规范
1. 文档结构
   - 使用 HTML5 文档类型声明
   - 设置正确的语言属性（zh-CN）
   - 包含必要的 meta 标签（charset, viewport）

2. 语义化标签
   - 使用 `<header>`, `<main>`, `<section>` 等语义化标签
   - 合理使用 `<div>` 进行布局分组

3. 表单元素
   - 为所有表单控件添加适当的 `id` 和 `name` 属性
   - 使用 `label` 标签关联表单控件
   - 为输入框添加合适的 `placeholder` 提示文本

### CSS规范
1. 变量定义
   - 使用 CSS 变量定义主题颜色
   - 变量命名采用语义化方式
   ```css
   :root {
       --primary-color: #4a90e2;
       --secondary-color: #f5f5f5;
       --text-color: #333;
   }
   ```

2. 响应式设计
   - 使用媒体查询适配不同屏幕尺寸
   - 采用移动优先的设计理念
   ```css
   @media (min-width: 768px) {
       /* 桌面端样式 */
   }
   ```

3. 布局方案
   - 使用 CSS Grid 和 Flexbox 进行布局
   - 避免使用固定像素值，优先使用相对单位

### JavaScript规范
1. 代码组织
   - 使用 DOMContentLoaded 事件确保 DOM 加载完成
   - 将相关功能组织在独立的函数中
   - 使用常量存储 DOM 元素引用

2. 事件处理
   - 使用事件委托优化性能
   - 为异步操作添加错误处理
   ```javascript
   try {
       await navigator.clipboard.writeText(text);
   } catch (err) {
       console.error('操作失败:', err);
   }
   ```

3. 数据处理
   - 使用正则表达式进行文本处理
   - 将复杂的替换规则抽取为配置对象
   ```javascript
   const symbolMap = {
       '，': ',',
       '。': '.',
       // ...
   };
   ```

## 功能实现指南

### 文本处理
1. 空格处理
   - 使用正则表达式 `/\s+/g` 匹配连续空格
   - 根据选项决定替换策略

2. 换行处理
   - 使用 `\n` 匹配换行符
   - 使用 `\n{3,}` 匹配多个连续换行

3. 特殊符号处理
   - 使用 Unicode 范围匹配中文字符
   - 使用映射对象处理符号转换

### 用户界面
1. 主题切换
   - 使用 CSS 变量实现主题切换
   - 将主题设置保存在 localStorage 中

2. 响应式设计
   - 使用 Grid 布局实现响应式界面
   - 为移动端优化交互体验

3. 交互反馈
   - 为按钮添加悬停效果
   - 提供操作成功的视觉反馈

## 性能优化建议

1. 代码优化
   - 缓存 DOM 查询结果
   - 使用事件委托减少事件监听器
   - 避免频繁的 DOM 操作

2. 用户体验
   - 添加加载状态提示
   - 优化大文本处理性能
   - 提供操作撤销功能

## 测试建议

1. 功能测试
   - 测试各种文本处理选项的组合
   - 验证特殊字符的处理结果
   - 检查主题切换功能

2. 兼容性测试
   - 在不同浏览器中测试
   - 验证移动端适配效果
   - 测试不同屏幕尺寸下的显示效果
